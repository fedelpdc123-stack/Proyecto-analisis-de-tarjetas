# Proyecto-analisis-de-tarjetas

**EN:** Credit card analysis project using SQL and Power BI.  
Built on a star schema with 3 related datasets (customers, cards, transactions).  
Focus on portfolio composition, customer behavior, and risk segmentation, etc.

Análisis en SQL y Power BI de modelo estrella de 3 datasets relacionados.

## Contexto
Fecha de referencia del análisis: 01/01/2026

## Datos
- 6.146 tarjetas de credito (cards_data)
- 2.000 clientes (users_data)
- 1.048.575 transacciones (transactions_data)

## Herramientas
Power BI Desktop, DAX, Power Query, SQL

## Páginas del Análisis en Power BI
1. Resumen

4. Límites de crédito

## Hallazgos principales
-Mastercard domina el mercado de las tarjetas
-Hay mas tarjetas de débito
-Hay gran cantidad de tarjetas vencidas
-La mayoria de las tarjetas tiene chip
-El límite de crédito promedio ronda los 15 mil
-Los PINs se actualizan cada 13 años en promedio
-2020 es el año con más aperturas
-El trimestre con más transacciones es el 3ro (julio, agosto, septiembre)
-El promedio de ingresos anuales de los clientes es de 45.720 dolares
-El 37,76% de los clientes son de alto riesgo
-El 46,50 % de las tarjetas tienen transacciones
-El 56,55% de los clientes hacen transacciones
